# Transformer

This folder contains experiments, implementations, and learning exercises related to Transformer architectures and GPT-style language models.

## Overview

The current notebook demonstrates core Transformer and GPT building blocks implemented from scratch using PyTorch. The goal of this project is to understand how modern autoregressive language models work internally rather than relying solely on high-level libraries.

Topics covered include:

* Token embeddings
* Positional embeddings
* GPT model structure
* Forward pass and logits generation
* Layer Normalization
* GELU activation function
* Transformer fundamentals
* Text generation concepts
* PyTorch-based implementations
